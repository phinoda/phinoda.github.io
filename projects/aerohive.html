BRANDING | UI

Aerohive

Geofencing service that helps store managers better monitor their stores by demonstrating API calls from Aerohive’s geofencing hardware.

Project Description

Modern networks must continuously adjust and adapt to keep up with the pace of change that mobility has created, something now unachievable in traditional network architectures.

Aerohive wants to create a new approach to adaptable, flexible, and cost effective wireless and wired access network. One of Aerohive’s practice is a product that helps store managers better monitor their stores by demonstrating API calls from Aerohive’s geofencing hardware.

I designed the responsive dashboard interface for the product.

Design Requirements

1. VIP Notification

When a VIP walks into the store, the store manager should receive a notification and the information of the VIP so that sales staff can give them A+ treatment and assistance.

2. Store Surge

A retail clothing store would like to be notified in advance of the need of additional checkout clerks so that she can open more registers by the time those shoppers are ready to pay.

3. VIP Notification

A list of geofenced devices should be shown in the app. If a device leaves the store range, the manager should receive a notification, and the device should be tagged as missing in the app.
